# Aegis

> An AI operating companion built from scratch.

## Vision

Aegis is a long-term personal AI project focused on creating an intelligent digital companion capable of:

- Understanding natural language
- Long-term memory
- Desktop automation
- Browser automation
- Voice interaction
- Computer vision
- Task planning
- Secure tool execution
